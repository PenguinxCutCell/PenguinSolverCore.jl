API Reference

Core System Hooks

PenguinSolverCore.rhs! — Function
rhs!(du, sys, u, p, t)

Semidiscrete residual evaluation hook. Physics packages are expected to implement this.

PenguinSolverCore.mass_matrix — Function
mass_matrix(sys)

Optional mass matrix for semidiscretizations. Return nothing for explicit ODE form.

PenguinSolverCore.apply_updates! — Function
apply_updates!(sys, u, p, t)

Optional hook to apply smooth time dependence and post-update bookkeeping.

PenguinSolverCore.rebuild! — Function
rebuild!(sys, u, p, t)

Optional hook to rebuild matrices/factorizations after discrete invalidations.

Updater Hook

PenguinSolverCore.update! — Function
update!(updater, sys, u, p, t) -> Symbol

Mutate sys (or p) in-place and return one of: :nothing, :rhs_only, :matrix, or :geometry.
